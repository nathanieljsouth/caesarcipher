• Caesar Cipher Instructions

New to the Caesar Cipher app? No problem, here's how to use it!

Overview

The Caesar Cipher app encrypts or decrypts a text string based on the 'cipher key'.
The key tells the app how many characters each letter in the input string should be moved forward.
For example, if you encrypted the string 'ABC' with a key of 5, the result would be 'FGH' because 'F' is 5 characters after 'A', 'G' is 5 characters after 'B' and so on.

This is what the Caesar Cipher app looks like - the coloured boxes around parts of the interface are explained below.


Encrypting Data

Step 1 - Type or paste some text into the 'input box' (shown in red in the image).
Instead of typing text in, you can click the 'quote of the day' button to quickly load today's quote into the input box. This is really useful for testing.

Step 2 - Chose a cipher key between 1 and 25 using the 'key' control (shown in green in the image). As you change the key you'll see the character map (shown in yellow in the image) update immediately.

Step 3 - Click the 'Encrypt!' button and the results will be shown in the output box (shown in blue in the image).

You can click the output box to immediately copy the results to the clipboard for easy pasting in another application.

Decrypting Data

There are two ways to decrypt an encrypted string:

If you know the cipher key, paste the encrypted string into the input box (shown in red in the image), select the correct cipher key (shown in green in the image) and click the 'Decrypt!' button. The decrypted string will be shown in the output box (shown in blue in the image).

If you don't know the cipher key, paste the encrypted string into the input box (shown in red) and then click the 'Brute Force Decrypt!' button. The app will decrypt the string using every possible cipher key and put all the results in the output box (shown in blue in the image). You'll need to look through all the results and figure out which one makes sense!

You can click the output box to copy the results to the clipboard for easy pasting in another application.
